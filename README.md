# Calculadora de División en C++

Este programa permite calcular:

- División real
- División entera
- Residuo entero (módulo)
- Residuo decimal

## Ejemplo

Entrada:
10 y 4

Salida:
División real: 2.5  
División entera: 2  
Residuo entero: 2  
Residuo decimal: 0.5
